# Sow Mkt Step 3 — Generate Messages

Generate 5 short, attractive marketing message variations for new product launches, new arrivals, and trending products.

---

## When to Use This Skill

Trigger this skill whenever the user wants short marketing copy for a product. Specifically:

- The user says any of these phrases: "write attractive message", "new product message", "write launch message", "new arrival just dropped", "create marketing message", "product promotion text", "trending product caption", "hype message", "promo line".
- The user provides a product name, category, or product link and asks for marketing copy, a caption, a tagline, a hook, or a hype message.
- The user mentions a launch, drop, sale, deal, discount, restock, or "back in stock" and wants short text to promote it.
- The user asks for Instagram, WhatsApp, SMS, push notification, banner, or email subject line copy for a product.

---

## What You Need

Required:
- Product name

Optional (ask only if missing AND helpful — never block on these):
- Category (e.g., skincare, dental, electronics)
- Key benefit (the #1 reason someone should buy it)
- Offer or discount (e.g., "20% off", "Buy 1 Get 1")
- Target audience (e.g., dentists, parents, gamers)

If the user only gives a product name, proceed anyway with smart defaults — do not over-ask.

---

## Steps

1. Read the user's product details (name, category, benefit, offer, audience).
2. If the product name is missing, ask once for it. Otherwise proceed.
3. Pick a distinct angle for each variation. Cover all five of these angles, one per variation:
   - **Excitement / launch** ("just dropped", "now live")
   - **Benefit / value** (the main reason to buy)
   - **Urgency / scarcity** ("only 10 left", "today only")
   - **Curiosity / hook** (a surprising or intriguing line)
   - **Social proof / trend** ("everyone's talking about it", "trending now")
4. Write each variation as a single line, max 1 line of text, simple English, with 1–3 relevant emojis.
5. Make sure each line highlights a benefit (not just a feature) and creates a feeling — excitement, FOMO, or curiosity.
6. Output exactly 5 variations in the format below. No extra explanation, no preamble.

---

## Output Format

Always reply with exactly this structure — nothing more, nothing less:

```
Variation 1:
<one-line message with emojis>

Variation 2:
<one-line message with emojis>

Variation 3:
<one-line message with emojis>

Variation 4:
<one-line message with emojis>

Variation 5:
<one-line message with emojis>
```

---

## Rules

Always do:
- Keep every message to **1 line only**.
- Use simple, clear English a 10-year-old could read.
- Add 1–3 relevant emojis per line for visual appeal.
- Focus on **benefits and feelings**, not technical features.
- Create urgency or excitement (words like "just dropped", "now live", "limited", "today only", "trending").
- Always produce **5 distinct variations** — never fewer, never more.

Never do:
- Never write paragraphs or multi-line messages.
- Never use jargon, buzzwords, or corporate-speak ("synergy", "leverage", "best-in-class").
- Never make false claims, fake discounts, or fake scarcity.
- Never copy a competitor's tagline.
- Never use more than 3 emojis in a single line.
- Never add a long explanation before or after the 5 variations.
